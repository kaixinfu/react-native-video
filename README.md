# react-native-videoDemo  <br/>
使用react native制作的一款网络音乐播放器 <br/>
基于第三方库 [ react-native-video ]( https://github.com/react-native-community/react-native-video ) 设计<br/>
"react-native-video": "^1.0.0"   <br/>

 - [x] 播放/暂停<br/>
 - [x] 快进/快退<br/>
 - [x] 循环模式(单曲,随机,列表)<br/>
 - [x] 歌词同步<br/>
 - [x] 进度条显示<br/>
 - [x] 播放时间<br/>
 - [x] 基本旋转动画<br/>
